I'm interested in the parts of computing where systems meet language models, where security meets data, and where small tools meaningfully change how people work. Most of what I ship is a developer tool, an ML pipeline, or something at the intersection.

What I'm working on

- [Pulse](https://github.com/bhumu105/pulse) — VS Code extension that tracks Claude Code cost, burn rate, and context health in real time. TypeScript, hook-based event capture.
- [Cue](https://github.com/bhumu105/cue.ai) — Chrome extension that scores prompt quality on claude.ai using a fine-tuned DistilBERT model running in the browser via ONNX Runtime. End-to-end ML pipeline (data labeling → training → ONNX export → deployment).
- [NetSentinel](https://github.com/bhumu105/network-anomaly-detector) — Network traffic anomaly detector using Isolation Forest and Random Forest on NSL-KDD-style features, with a Flask dashboard and live Scapy capture.
